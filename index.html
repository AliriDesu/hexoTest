DOM

描述

提供了一套用来管理控制，操作html页面的规则
是将页面中的原色，文本等都看做是节点，并由节点构成树状结构（DOM Tree）
通过管理DOM Tree来间接达到操作管理页面的目的。

DOM API

1. node.nodeName    节点名称
2. node.nodeType    节点类型

DOM Node API

规定

DOM中为了提高性能，规定如果可以获取到DOM Tree中的某一个节点，那么就可以通过这个节点直接或简介的访问到DOM Tree中的任何一个节点，可以有效的提高浏览器性能

说明

HTML页面加载的过程中存在两个严重问题，【回流reflow】和【重绘repaint】

回流 reflow

页面中元素的布局、尺寸等样式发生变化导致的页面重新加载

重绘 repaint

页面中元素的颜色或者文字颜色等样式发生变化导致的页面重新渲染。
  1. 返回当前节点的父节点 node.parentNode
  2. 返回当前节点的子节点 node.childNodes
  3. 返回当前节点的前一个兄弟节点 node.previousSibling
  4. 返回当前节点的后一个兄弟节点 node.nextSibling

    扩展

    2. 合理使用文档碎片节点 documentFragment

18-JS-事件

发表于 2021-08-13

事件

描述

是一个抽象的过程，看不见，摸不到
js中为事件监听的回调函数内部，添加了一个名为event的时间对象，这个对象中包含了时间发生时候的所有特征。

说明

不同的事件，它的事件对象中的属性也不相同。

补充

时间对象使用形参的方式调用，会存在某些时间对象属性不生效的情况
使用系统提供event对象，则永远生效

语法

event

监听

描述

当某件事情发生的时候，能够对应做出响应的操作

语法

HTML级 事件监听

描述

简称为html事件，是指添加事件监听的操作发生在html代码中的监听

语法

<E on+事件名="函数名;"></E>

缺陷

耦合性过高，不利于代码的模块化

DOM0级 事件监听

描述

简称为DOM0事件
是指在js脚本中，通过.on添加的监听

语法

node.onclick = function(){

}

缺陷

对于【同一个节点】，【同一个事件】类型，添加的多个监听会产生覆盖，以最后一次添加为准

DOM2级 事件监听

描述

通过指定函数，添加或移除的事件监听

语法

添加：
    node.addEventListener(事件名，外部参数名|回调函数，？useCapture);
添加：
    node.removeEventListener(事件名，外部参数名，？useCapture);

说明

DOM2事件监听的一出操作必须保证移除函数的代码参数和添加时的参数完全一致

优点

结构明晰，不会产生覆盖

事件类型

类型

鼠标事件

  1. 单击事件 click
  2. 双击事件 dblclick
  3. 按下事件 mousedown
  4. 抬起事件 mouseup
  5. 进入事件 mouseenter | mouseover
  6. 离开事件 mouseleave | mouseout
  7. 移动事件 mousemove

    键盘事件

  8. 按下事件 keydown | keypress
  9. 抬起事件 keyup

    焦点事件

  10. 获得焦点 focus
  11. 失去焦点 blur

    滚动事件

    元素滚动 页面滚动 scroll

    加载时间

    页面加载 元素加载 load

    尺寸变化事件

    页面尺寸变化 元素尺寸变化 resize

    事件传递

    描述

    在DomTree中，事件当在某一个节点上被触发的时候，会优先和当前触发节点进行交互，在触发了当前节点的事件监听后，并不会立即消失，而是会沿着DomTree向上传递，并沿途触发经过的每一个节点上的事件监听，直到document节点为止。

    类型

    冒泡传递

    事件由当前触发节点开始，沿着DomTree逐级向上传递，直到document节点为止

    捕获传递

    事件由document节点开始，沿着DomTree逐级向下传递，直到触发节点为止

    停止传递

    event.stopPropagation()

    事件委托

    描述

    时间监听并不会直接添加在当前节点，而是会添加在当前节点的父节点上，由父节点同意调配需要执行的内容。这种将时间监听处理推迟到父节点中的操作称为事件委托。

    原理

    利用了【事件传递】+【event.target】

公有和私有

描述

公有：对象外部能够直接访问到的属性和方法，就成为共有属性或公有方法
    实例属性：类中对象所共同具备，但属性值不同的属性
    公共属性：类中所有对象共同拥有的
私有：对象外部不能够直接访问到的属性和方法，就成为私有属性或私有方法

说明

js中不存在公有和私有的概念
但js使用【this.xxx】和原型【prototype】来模拟了公有
通过【局部变量】来模拟了私有

原型prototype （类中的一坨空间，这个空间被类的所有对象共同拥有）

描述

js为函数（构造函数）提供了一个对象属性
所有添加在这个对象类型的属性中的内容，能够被所有通过这个创建的对象所共同拥有

语法

函数语法

对象语法

构造函数实例.__proto__
ps：__proto__原型指向，只想了这个对象的类的原型

js对象属性访问原则

描述

js中，访问对象的属性的时候，优先去对象内部查找，如果找得到，认为对象拥有这个属性
但如果没找到，那么就去【创建了这个对象的类的原型】中查找
如果找得到，也认为对象拥有这个属性
如果仍然没找到，那么就去【创建了 创建了这个对象的类的原型 的类的原型】中查找
...
如果一直到Object.prototype都没能找到这个属性，就认为对象不具备这个属性

继承

描述

15-JS-函数

发表于 2021-08-09

函数

描述

1. 具有某些特定功能，可以重复调用，闭合的代码结构

语法

函数定义

function 函数名 (形式参数1，形式参数2，...){
    函数体() 
}

函数调用

函数名称(参数1，参数2，...)

说明

1.  函数名：等同于变量名
2.  函数如果只声明，不调用，那么函数相当于不存在
3.  形式参数：形参指的是在定义函数的时候，写在小括号内部，用来表示函数在执行的时候，将要被传入真正的参数的一个载体
4.  实际参数，实参指的是在调用函数的时候，写在小括号内部，真正参与运算的内容
5.  不定参函数：
    1.  定义时的参数数量大于调用时的数量
        通常不建议使用，因为属于非法使用
        如果必须要使用，一般通过使用【参数初始化】来避免错误
    2.  定义时的参数数量小于调用时的参数数量
        js中函数的一种正常使用规则
        利用js提供给函数内部使用的名为arguments对象的结构
        其包含了函数在调用的时候的所有参数
6.  返回值
    是在函数内部，通过return关键字定义的，用于表示函数的执行结果的，能够将这个结果返回到代码中函数世纪调用的位置，并代替函数调用语句的结构
    return表示函数执行结束

扩展

1.  一等公民
    只要变量可以存在的地方，函数就可以存在
2.  基本数据类型和复杂数据类型在函数传递参数的时候，有不同的表现形式
    值传递和引用传递（地址传递）

14-JS-数组、条件与循环

发表于 2021-08-06

数组 Array

描述

数组是一种数据类型，可以认为是一种非常特殊的对象。
特殊之处在于其属性是数字类型。

语法

js中数组是使用[]定义的，中括号内部的内容由逗号隔开的，内容可以是任意数据类型的数据结构

名词

元素

数组内容使用逗号隔开的内容称为是数组元素

下标

系统自动添加的，从零开始的，表示数组元素序号的标记

操作

读

获取数组中的元素内容
arr[index]

写

设置活添加数组中元素的内容
arr[index] = xxx

注意

数组的下标是系统自动添加的连贯的数字。如果给定的下标>实际的最大下标，数组的长度会被拉长。

条件

描述

表示代码不再从上到下按照顺序执行，而是会经历判断后决定执行的走向

语法

if(带有逻辑值的表达式){
    当逻辑表达式的值为true的时候需要执行的代码;
}

注意

当if结构的逻辑表达式的结果是false的时候，整个if结构都相当于不存在
这种if结构叫【一元判断】

变化

if-else

描述

本结构是为了弥补if条件判断的一元特性而提出的，本结构为二元判断

if-else if-else

描述

本结构是为了弥补if条件判断的一元特性而提出的，本结构为二元判断

循环

描述

不需要重复书写大量冗余代码来具体描绘每次要做的事，而是通过书写规则，来保证代码按照规则重复执行，知道规则被打破为止，循环中止。

语法

for(表达式1;表达式2;表达式3){
    需要重复执行的代码
}

说明

表达式1：循环变量赋初值
表达式2：循环中止条件（循环得以继续执行的条件）
表达式3：循环变量向着循环终止条件的变化趋势

js说明

描述

js是一门面向过程的脚本式编程语言
    面向过程 没有【对象】这种数据结构
    面向对象 包含【对象】这种数据结构

说明

js为了能够与其他的编程语言协同工作，达到脚本语言的能力，不得不使用某种【已经存在的数据结构】来模拟【面向对象语言中的对象结构】

js对象

描述

在js中，对象是通过大括号{}定义的，在大括号内部使用键值对构成的，键值对之间用逗号间隔的，键名与键值之间使用冒号间隔的数据结构。

语法

var obj = {key1:value1,key2:value2,...}

说明

1.  key（键）：也叫键名、属性、属性名、方法、方法名，要求必须是字符串类型
2.  value（值）：也叫键值，方法、方法、属性值，可以是任意数据类型

操作

读取操作 获取对象属性的值
    对象名.属性名
    对象名['属性名']
写入操作 修改或创建属性的值
    同上↑

补充

1.  对象的属性可以不加引号，但不写引号时属性必须符合变量命名规则
2.  js中对象可以认为有三大类
    1.  用户自定义对象
    2.  系统内置对象
    3.  主机环境下的浏览器对象

Number数字对象

API
    1. toFixed(x)
        描述：数字转换字符串，四舍五入，保留x位小数，x默认是0

String字符串对象

说明：
    下标：字符串中字符的序号，下标从0开始。下标系统自动添加
    长度：字符串中字符的个数
API
    1. slice(start,end)
        描述：截取字符串的片段并返回
        说明：
            1.  截取的时候从start到end，但不包括end序号位置的字符
            2.  截取无论正数序号还是负数序号都是从左到右
            3.  如果第二个参数不谢，默认从开始截取下表为止一直到末尾

Date对象

描述

Date对象在js中表示日期的含义是Object类型，但经常表现为字符串的样子

语法

var date = new Date(); //获取当前时间

var date2 = new Date(YYYY,MM,DD,HH,mm,ss); //创建指定日期，月份从0开始

作用

        加法：字符串拼接
        减法：毫秒数的差值

本质

日期在内存中的存储是【距离1970年1月1日0点0分0秒0毫秒】的毫秒数

1. 当运算符的左右两端的运算子均为数字类型时，仍采用基本数学运算规则
2. 当运算符的左右两端的运算子不都是数字类型时
   1. 加法会先将非字符串类型转换成字符串类型，然后再将两个字符串拼接
   2. 其余会先将字符串转换成数字类型，再按照数学逻辑运算。当字符串无法转换成数字类型的时候，运算结果就会出现NaN

扩展 三目运算符：?:

语法
表达式1 ? 表达式2 : 表达式3
名词
表达式-> 具有含义的式子
作用
判断表达式1的逻辑值，真则结果为表达式2，假则结果为表达式3

名词概述

说明：JavaScript 本名 LiveScript
标准：ES5 ES6 
    js无法直接操作html页面，因为js操作的是内存，如果想要操作页面，则必须借助于某种手段来间接操作页面。这个手段就是dom。
名词：
    DOM：文档对象模型。一套用于帮助js管理操作html页面的规则。
        dom中提供了非常多的api，通过api可以打到操作页面的目的。
    API:应用程序接口。提供了一些固定的途径。通过这些途径可以完成对封闭
        系统内部的某些功能，却不需要了解内部是如何实现的。
    节点：dom规则中认为，页面的元素就是节点，文本也是节点（node）。
    节点树：dom tree，由节点构成的树状结构。
    根节点：document节点。是整个节点树的根节点。表示整个页面本身。

dom规则

api：

1. 选中页面元素
    语法：
         document.querySelector('将要选中的页面元素的选择器名称');
2. 给选中的元素添加点击事件
    语法：
        内存名称.onclick = function () {
            ...
        }
3. 给选中的元素修改内容
    语法：node.innerHTML （不可编辑标签）| node.value = xxx （可编辑标签）;
4. 给选中的元素修改样式
    语法：node.style.xxx = css样式值 | node.style.cssText = css代码
    如果出现中划线，将中划线删除，将后续的首字母大写。
5. 在内存中创建新的元素，并添加至页面中
    语法：document.createElement('创建的元素名称');
    node.appendChild(元素);

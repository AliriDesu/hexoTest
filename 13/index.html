13-JS-对象概述

发表于 2021-08-04 更新于 2021-08-05

js说明

描述

js是一门面向过程的脚本式编程语言
    面向过程 没有【对象】这种数据结构
    面向对象 包含【对象】这种数据结构

说明

js为了能够与其他的编程语言协同工作，达到脚本语言的能力，不得不使用某种【已经存在的数据结构】来模拟【面向对象语言中的对象结构】

js对象

描述

在js中，对象是通过大括号{}定义的，在大括号内部使用键值对构成的，键值对之间用逗号间隔的，键名与键值之间使用冒号间隔的数据结构。

语法

var obj = {key1:value1,key2:value2,...}

说明

1.  key（键）：也叫键名、属性、属性名、方法、方法名，要求必须是字符串类型
2.  value（值）：也叫键值，方法、方法、属性值，可以是任意数据类型

操作

读取操作 获取对象属性的值
    对象名.属性名
    对象名['属性名']
写入操作 修改或创建属性的值
    同上↑

补充

1.  对象的属性可以不加引号，但不写引号时属性必须符合变量命名规则
2.  js中对象可以认为有三大类
    1.  用户自定义对象
    2.  系统内置对象
    3.  主机环境下的浏览器对象

Number数字对象

API
    1. toFixed(x)
        描述：数字转换字符串，四舍五入，保留x位小数，x默认是0
        语法：var str = number.toFixed(x)

String字符串对象

说明：
    下标：字符串中字符的序号，下标从0开始。下标系统自动添加
    长度：字符串中字符的个数
API
    1. slice(start,end)
        描述：截取字符串的片段并返回
        说明：
            1.  截取的时候从start到end，但不包括end序号位置的字符
            2.  截取无论正数序号还是负数序号都是从左到右
            3.  如果第二个参数不谢，默认从开始截取下表为止一直到末尾

Date对象

描述

Date对象在js中表示日期的含义是Object类型，但经常表现为字符串的样子

语法

var date = new Date(); //获取当前时间
var date2 = new Date('YYYY-MM-DD HH:mm:ss'); //创建指定日期
var date2 = new Date(YYYY,MM,DD,HH,mm,ss); //创建指定日期，月份从0开始

作用

运算：日期支持减法和减法
        加法：字符串拼接
        减法：毫秒数的差值

本质

日期在内存中的存储是【距离1970年1月1日0点0分0秒0毫秒】的毫秒数

JSON

描述

JSON是一套【利用了JS对象语法的】数据传输格式规范。

语法

JSON无法直接写出来，必须通过API进行转换

API

将非JSON转换为JSON（编码）
    var jsonStr = JSON.stringify(非JSON数据)
将JSON转换回非JSON（解析）
    var 数据 = JSON.parse(JSON数据)
